- 这个课程的hw会实现很多op和操作，不过我觉得最精髓的地方是以下几点
1. 一个个tensor结点通过op联系起来构造一张计算图
2. 由op的backward操作实现的自动微分
3. 通过NDArray对象提供了三种不同的底层数组类型
4. 同时还有一些trick很有意思，比如通过shape，stride和offset就可以将一个基础的C语言中的数组操作得和numpy中的数组一样灵活
